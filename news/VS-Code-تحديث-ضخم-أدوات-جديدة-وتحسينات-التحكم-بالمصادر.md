أصدرت Visual Studio Code تحديثها الجديد 1.101 الذي يركز بشكل كبير على تعزيز التكامل مع Model Context Protocol (MCP)، حيث يتيح التحديث الجديد استخدام أوامر الشرطة المائلة للوصول إلى مطالبات MCP من خلال نافذة الدردشة، مع دعم مقتطفات التعليمات البرمجية القابلة لإعادة الاستخدام والإكمال التلقائي. كما أصبح من الممكن حفظ الموارد وإرفاقها واستعراضها مباشرة من واجهة الدردشة باستخدام أوامر مخصصة. يقدم التحديث أيضاً تجربة "العيّنات التجريبية" التي تسمح لخوادم MCP بتقديم طلبات "استدعاء" لنماذج اللغة، مع توفير ضوابط وصول وتسجيل. كما يضيف التحديث دعماً لبروتوكولات المصادقة الجديدة، بما في ذلك التسجيل الديناميكي للعملاء.

بالإضافة إلى ذلك، يأتي التحديث بتحسينات ملحوظة على سير عمل الدردشة، من خلال إضافة مجموعات أدوات لتجميع الأدوات ذات الصلة، وإمكانية الوصول إليها عبر علامات التصنيف أو من خلال واجهة المستخدم. كما أصبح بإمكان المستخدمين تخصيص أوضاع الدردشة لتحديد الوصول إلى الأدوات وسلوكها لمهام سير عمل محددة. وقد تم تحسين سهولة الاستخدام عبر فصل الرسائل بشكل أوضح، وإظهار إجراءات التراجع بشكل مرئي، وتحسين التنقل بين المرفقات. وأخيراً، حصل نظام التحكم في المصادر على عرض بياني (Graph view) وتكامل مع GitHub Copilot لتخصيص المهام، بالإضافة إلى خيار "الإصلاح مع Copilot" لأخطاء تكوين المهام.
